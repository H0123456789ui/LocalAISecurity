# 更新日志

## [1.0.0] — 2026-05-29

### ✨ 新增
- **全盘杀毒扫描** — 一键扫描进程/网络/启动项/可疑文件，结果分4类展示
- **C盘空间总览** — 彩色进度条显示系统/软件/用户/其他/可清理文件占用
- **分类详情窗口** — 点击分类查看具体文件列表，18级重要性标注，可筛选可清理项
- **超大文件扫描** — 自动发现 >100MB 文件，标注可删性
- **扫描进度条** — C盘分析全程显示确定进度，每2000文件更新
- **一键清理安全项** — C盘扫描后一键清理AI确认安全的文件
- **启动项扫描** — 检查注册表Run/RunOnce键和启动文件夹
- **共享算子库** (`nn_ops.h`) — 消除约200行重复代码
- **模块级签名验证** (`_verify_signature_winapi`) — 可复用
- **用户安全列表** — 可手动添加信任进程

### 🔧 修复
- 修复 `extract_features` 中静态变量导致的线程安全问题
- 修复 `LocalDataManager` 统计数据读写竞态条件
- 修复硬编码路径 → 动态环境变量（SystemRoot/SystemDrive）
- 修复 RegistryMonitor → 每键独立线程并行监控
- 修复 C盘分析器递归扫描被截断（深度限制+锁定目录跳过+连接点死循环）
- 修复数字签名验证从 PowerShell 子进程 → ctypes WinVerifyTrust

### 🏗 优化
- C盘分析改用 `os.walk` + `os.scandir` 快 3-5 倍
- 添加 Windows 连接点（junction）检测和跳过，防止死循环
- 扫描增加 120 秒超时保护
- 报告间隔从 5000 降到 2000 文件，进度反馈更细腻
- net_update_module 内置零依赖 JSON 解析器
